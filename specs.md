# Техническое задание: CSV Median Calculator

В этом файле размещена краткая версия технического задания для тестового задания C++ разработчика.

Полное описание задания (структура входных CSV, формат конфигурации, требования к поведению приложения, критерии оценки и т.д.) предоставлено в задаче, сопровождающей этот репозиторий.

Ключевые требования:
- Язык: C++23
- Сборка: CMake >= 3.23
- Зависимости: Boost.Program_options, Boost.Accumulators, toml++, spdlog (подключаются через FetchContent / Boost find_package)
- Приложение читает CSV-файлы из директории, сортирует по `receive_ts`, инкрементально рассчитывает медиану цен и записывает изменения медианы в CSV-выход.

Файлы в репозитории:
- `CMakeLists.txt` — конфигурация сборки
- `src/` — исходники
- `config.toml` — пример конфигурации
- `examples/input/` — пример входных CSV
- `examples/output/` — директория для результатов
